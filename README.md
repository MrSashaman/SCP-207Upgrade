# SCP-207Upgrade — SCP:SL LabAPI Plugin

A plugin for **SCP: Secret Laboratory (LabAPI)**. It will add new features to SCP-207 on your server.

## Modifications:
- 💣 Kamikaze Upgrade
- 🌹 Regen Upgrade
- 🏃‍♀️ Speed Upgrade 

---

## Installation:

### 1. Download the latest version of the plugin from:
```bash
https://github.com/MrSashaman/SCP-207Upgrade/releases
```
### 2. Move the file to the specified path:
```bash
C:\Users\AppData\Roaming\SCP Secret Laboratory\LabAPI\plugins\global
```
### 3. Run server
